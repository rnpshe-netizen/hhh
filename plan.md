# 프로젝트 설계도 (Plan)

## 1. 프로젝트 목적
- 집과 회사 양쪽 환경에서 끊김 없이 코드를 공유하고 개발할 수 있도록 Git(버전 관리) 및 원격 저장소 연동 환경을 구축합니다.

## 2. 요구사항
- Git 프로그램 설치 유무 확인 및 최초 세팅 (이름/이메일 등)
- GitHub 계정 생성 및 저장소(Remote Repository) 연결

## 3. 전체 구조 및 기술 스택
- **버전 관리 시스템:** Git
- **원격 저장소 플랫폼:** GitHub

## 4. 진행 계획
1. **[기본 세팅]** 관리자님 PC에 Git이 설치되어 있는지 확인하고, 안 되어 있다면 설치를 안내해 드립니다. 설치 후 작업자 이름과 이메일을 등록합니다.
2. **[내 확인]** 관리자님께서 GitHub 사이트에 가입하고 새 '빈 창고(Repository)'를 하나 생성해 주시면 그 주소를 받습니다.
3. **[기능 연결]** 현재 작업 폴더(`C:\HHH`)를 Git 로컬 창고로 만들고(`git init`), 방금 생성하신 GitHub 클라우드 창고와 동기화 라인을 연결합니다.
4. **[내 확인]** 지금까지 정리된 문서를 첫 번째로 저장(Commit)하고 클라우드에 업로드(Push)하여 최종 확인을 합니다.
