## 매니저님 구글 폼 생성 순서
1. Google Forms (forms.google.com) 접속
2. 위 설계대로 폼 생성
3. 응답 탭 → 스프레드시트 연결 → 시트 ID 복사
4. 시트 ID를 Vercel 환경변수에 등록
